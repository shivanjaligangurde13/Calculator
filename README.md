A simple and responsive mini calculator built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Addition, Subtraction, Multiplication, Division
- Interactive UI
- Responsive design
- Easy to use

## 🛠️ Tech Stack

- CSS
- JavaScript
